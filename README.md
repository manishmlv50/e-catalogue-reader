# E-Catalogue Reader

Overview
--------
e-catalogue reader is a small utility for opening, parsing, and browsing electronic product catalogues. It is designed to help developers and users inspect catalogue data, search products, and export or integrate product lists into other systems. The project is intentionally lightweight and format-agnostic (common catalogue formats: CSV, JSON, PDF).

Prerequisites
-------------
- A runtime matching the project (check repository for package.json or requirements.txt)
- Git (to clone the repository)
- Optional: Node.js (if package.json exists) or Python 3.8+ (if requirements.txt exists)

Quick start
-----------
1. Clone the repository:
   git clone <repo-url>
   cd e-catalogue-reader
2. Browse the index.html to view the catalogues in the flipbook
